# 超市管理系统 java175

## 项目概述

本项目是一款基于B/S架构的超市管理系统，采用MVC模式设计，结合了Java、SpringBoot、Vue等现代技术栈进行开发。系统实现了前后端分离，具有高可维护性和可扩展性。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**:
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Redis
  - Vue
